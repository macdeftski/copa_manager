# copa_manager
Sistema de gestão de Seleções, Jogadores e Partidas no contexto de Copa do Mundo de Futebol.
O programa consiste de um menu e vários sub-menus, um para Seleções, um para Jogadores e um para Partidas.
Para utilizar o programa, selecione o sub-menu que você deseja utilizar, e dentro do sub-menu selecione a opção que deseja.
Em cada menu, o usuário pode criar, listar, ordenar e filtrar os itens.
O programa foi feito em um loop eterno, até que o usuário peça para sair.
